# 5xSun - AutoRefill

**5xSun - AutoRefill** is a mod for *Surviving Mars Relaunched* that automatically scans and refills resource deposits on your map. 

## Features
- **Automated Refill:** Automatically replenishes surface and subsurface deposits (Concrete, Metals, Rare Metals, Water, and Rare Minerals) to their maximum capacity.
- **Smart Threshold:** By default, it only refills deposits that fall below a 30% threshold.
- **Symmetrical Scheduling:** Scans the map 5 times a day (at hours 0, 5, 10, 15, and 20) to ensure a steady supply of resources.
- **In-Game Notifications:** Receive non-intrusive on-screen notifications summarizing how many deposits were refilled and which resources were affected.
- **Customizable:** Fully synchronizes with in-game Mod Options so you can tweak the refill threshold and toggle specific resources on or off.

## Compatibility
Compatible with *Surviving Mars Relaunched*. It dynamically supports all loaded realms (Surface, Subsurface, Asteroids) and safely falls back to standard APIs if needed.

## Installation
Subscribe to the mod through the Paradox Mods portal or place it in your local mods directory. Activate it in the Mod Manager before loading a save or starting a new game.
